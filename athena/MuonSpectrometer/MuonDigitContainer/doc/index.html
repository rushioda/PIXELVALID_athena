MuonDigitContainer

Version 0.0
11sep02 1650 EDT

This package provides containers for muon digits. They may be filled directly or (eventually) from raw data, zebra and standard StoreGate formats such as objectivity and root.

The design for this package may be found at http://www.usatlas.bnl.gov/~dladams/mudigi.

Status

The transient part of the MDT digit container is implemented.

Components

The components here are listed in dependency order. Each component depends only on those listed above it.

MdtDigit (header, source, test)

Temporary description of an MDT digit. This class should be moved to another package.

MdtDigitContainer (header, source, test)

Container for MDT digits.

RpcDigit (header, source, test)

RPC digit.
