How to try a test run of LArROD :

  1. Do the following shell setup (probably the least that is possible) : 

        setenv ATLAS_ROOT /afs/cern.ch/atlas/software
        setenv CVSROOT $ATLAS_ROOT/cvs
        setenv PATH "$PATH":"$ATLAS_ROOT"/bin
        source `srt setup -s csh` 
  2. setenv SRT_RELEASE 1.3.2    ( to setup for the release 1.3.2; otherwise replace 1.3.2 with the release version that you want to use)
  3. Pick a build directory and then do
    srt new src $SRT_RELEASE
  4. cd src
  5. cvs checkout -d LArROD offline/LArCalorimeter/LArROD
  6. cd ..
  7. mkdir build
  8. cd build
  9. ../src/configure
  10. make clean       ( NOT necessary after the first time )
  11. make install
  12. cd ..
  13. mkdir run
  14. cd run
  15. source ../src/LArROD/misc/runme.csh
  16. athena

And you see it running !
